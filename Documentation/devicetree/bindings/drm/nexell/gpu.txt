Device-Tree bindings for Nexell's DRM and display control driver

Required properties for display controller driver:
 - compatible: value should be "nexell,dp-s5p6818"

Required properties for LCD output driver:
 - compatible: value should be "nexell,dp-s5p6818-lcd"

Required properties for DRM driver:
 - compatible: value should be "nexell,drm-s5p6818"

Required properties for DRM sub LCD driver:
 - compatible: value should be "nexell,drm-s5p6818-lcd"

Example:

/ {
	...

	soc {
		dp-controller@N {
			compatible = "nexell,dp-s5p6818";
		};

		dp-controller@1 {
			compatible = "nexell,dp-s5p6818";
		};

		dp-out-lcd {
			compatible = "nexell,dp-s5p6818-lcd";
		};

		drm-subsystem {
			compatible = "nexell,drm-s5p6818";
		};

		drm-sub-lcd {
			compatible = "nexell,drm-s5p6818-lcd";
		};

};
